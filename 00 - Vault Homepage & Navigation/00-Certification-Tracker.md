# 00-Certification-Tracker

> [!abstract] Overview
> A comprehensive tracking matrix and preparation resource for the essential certifications required for a Desktop Support Engineer (DSE). This note details exam domains, study milestones, self-assessment scoring, and resources for CompTIA A+ (Core 1 & Core 2), ITIL v4 Foundation, and Microsoft 365 Certified: Endpoint Administrator Associate (MD-102).

---

## Why Certification Matters in Desktop Support
Certifications validate your technical knowledge to employers and provide a structured learning path. 
*Seedha simple shabdon mein bole toh: Ek formal certificate hone se recruiters ko trust hota hai ki aapko IT operations ki standard knowledge hai. Ye aapke resume ko filter se nikalne aur higher salary range secure karne mein help karta hai.*

## 1. CompTIA A+ (Core 1: 220-1101 & Core 2: 220-1102)
CompTIA A+ is the industry standard for establishing a career in IT. It consists of two exams. You must pass both to become certified.

### Exam Details
- **Passing Score:** 675 / 900 (Core 1) and 700 / 900 (Core 2)
- **Number of Questions:** Max 90 questions per exam (multiple-choice and performance-based drag-and-drop tasks)
- **Cost:** ~$250 USD per voucher (varies by region)

### Core 1 (220-1101) Domain Checklist
- [ ] **Domain 1.0: Mobile Devices (15%)**
  - [ ] Laptop hardware installation (RAM, storage, keyboards)
  - [ ] Mobile display components (LCD, OLED, Digitizers, Webcams)
  - [ ] Mobile connectivity (Cellular, Wi-Fi, Bluetooth, NFC)
- [ ] **Domain 2.0: Networking (20%)**
  - [ ] Network ports, protocols, and standard services (TCP/UDP)
  - [ ] Routers, switches, access points, and firewall setup
  - [ ] IP addressing configurations (IPv4/IPv6, DHCP, Static IP)
- [ ] **Domain 3.0: Hardware (25%)**
  - [ ] Cable types, connectors, and interface standards (USB, HDMI, DisplayPort)
  - [ ] System memory types (DDR4, DDR5, SODIMM)
  - [ ] Storage devices (SATA, NVMe PCIe, RAID configurations)
  - [ ] Motherboard form factors, CPU sockets, cooling methods, and BIOS/UEFI setup
- [ ] **Domain 4.0: Virtualization and Cloud Computing (11%)**
  - [ ] Cloud concepts (SaaS, IaaS, PaaS, public vs. private clouds)
  - [ ] Client-side virtualization (hypervisors, VM configuration)
- [ ] **Domain 5.0: Hardware and Network Troubleshooting (29%)**
  - [ ] Troubleshooting Motherboards, RAM, CPU, and Power Supplies
  - [ ] Troubleshooting Storage systems and RAID arrays
  - [ ] Troubleshooting display and peripheral issues
  - [ ] Troubleshooting wired and wireless network connections

### Core 2 (220-1102) Domain Checklist
- [ ] **Domain 1.0: Operating Systems (31%)**
  - [ ] Windows features, installation, upgrade paths, and administrative tools
  - [ ] Command-line tools (CMD, PowerShell)
  - [ ] OS directory structures, system utilities, control panel tools
  - [ ] macOS and Linux operating system features
- [ ] **Domain 2.0: Security (25%)**
  - [ ] Physical security controls (locks, badges, biometrics)
  - [ ] Logical security (Active Directory, GPO, permissions, MFA)
  - [ ] Wireless security protocols (WPA2, WPA3)
  - [ ] Malware detection, removal, and prevention methods
- [ ] **Domain 3.0: Software Troubleshooting (22%)**
  - [ ] Troubleshooting Windows startup issues, BSODs, and registry errors
  - [ ] Troubleshooting mobile OS application errors and sync issues
  - [ ] Security incident response (remediating compromised systems)
- [ ] **Domain 4.0: Operational Procedures (22%)**
  - [ ] Best practices for safety, ESD mitigation, and environmental controls
  - [ ] Professional communication, active listening, and etiquette
  - [ ] Documentation, change management, and asset tracking

## 2. ITIL v4 Foundation (IT Service Management)
ITIL v4 focuses on modern IT service delivery, combining traditional IT service management with Agile, DevOps, and Lean practices.

### Exam Details
- **Passing Score:** 26 / 40 (65%)
- **Format:** 40 multiple-choice questions, closed book
- **Cost:** ~$350 - $450 USD (often includes course materials)

### Key Exam Topics Tracker
- [ ] **The Four Dimensions of Service Management** (Organizations & People, Information & Technology, Partners & Suppliers, Value Streams & Processes)
- [ ] **The ITIL Service Value System (SVS)** (Opportunity, Demand, Value, Guiding Principles, Governance, Service Value Chain)
- [ ] **The 7 ITIL Guiding Principles** (Focus on value, Start where you are, Progress iteratively with feedback, Collaborate and promote visibility, Think and work holistically, Keep it simple and practical, Optimize and automate)
- [ ] **ITIL Practices**
  - [ ] **Incident Management** (Resolving outages quickly)
  - [ ] **Problem Management** (Identifying root cause of incidents)
  - [ ] **Change Enablement** (Planning changes to minimize risk)
  - [ ] **Service Request Management** (Handling requests like access, hardware upgrades)
  - [ ] **Service Desk** (Single point of contact for users)

## 3. Microsoft 365 Certified: Endpoint Administrator Associate (MD-102)
MD-102 is the premium certification for modern endpoint management, replacing the older MD-100 and MD-101. It is essential for advanced support engineers managing corporate fleets.

### Exam Details
- **Passing Score:** 700 / 1000
- **Cost:** ~$165 USD
- **Key Concepts:** Microsoft Intune, Windows Autopilot, Azure AD, Co-management, Defender for Endpoint.

### Domain Checklist
- [ ] **Deploy Windows client (25-30%)**
  - [ ] Plan and execute Windows upgrades using Autopilot
  - [ ] Configure custom OS installation and provisioning packages
  - [ ] Deploy client-side configurations (languages, apps)
- [ ] **Manage identity and compliance (15-20%)**
  - [ ] Manage user accounts and Group Membership in Entra ID (Azure AD)
  - [ ] Configure conditional access policies and MFA requirements
  - [ ] Monitor device health and compliance reports in Intune
- [ ] **Manage, maintain, and protect devices (40-45%)**
  - [ ] Device enrollment in Microsoft Intune (MDM/MAM)
  - [ ] Configure device profiles, GPO migrations, and security baselines
  - [ ] Deploy software updates, driver updates, and patches
  - [ ] Implement Defender for Endpoint protection policies
- [ ] **Manage applications (10-15%)**
  - [ ] Deploy application packages (MSI, Win32, Store apps) via Intune
  - [ ] Configure application protection policies and licenses

## Senior Engineer Study Tips
> [!tip] From the Field
> 1. **Focus on PBQs (Performance Based Questions):** In CompTIA A+, these carry the most weight. Practice configuring SOHO routers, interpreting command prompt outputs (`ipconfig`, `ping`, `tracert`), and dragging hardware components to correct motherboard slots.
> 2. **Spaced Repetition:** Use Anki flashcards for learning network port numbers (e.g., DNS=53, HTTPS=443, RDP=3389) and event viewer IDs.
> 3. **Build a Lab:** Never study dry concepts. Install Oracle VM VirtualBox and run Windows Server (evaluation version) to create a domain controller and configure Active Directory yourself.
