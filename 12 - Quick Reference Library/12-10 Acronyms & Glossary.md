# 12-10 Acronyms & Glossary

> [!abstract] Overview
> An alphabetical reference glossary containing essential technical acronyms and terms utilized in corporate desktop support, IT service management (ITSM), and networking environments.

---

## 1. What Is It? (Concept Explanation)
This glossary is a reference dictionary designed to standardize communications and terminology for IT support professionals. Standard definitions help align technicians with enterprise operational workflows.
*Seedha simple shabdon mein: IT industry acronyms (jaise AD, DNS, SLA, GPO) se bhari hui hai. In technical terms ko samajhna aur sahi context mein use karna ek competent desktop support engineer ki pehchan hai. Yeh glossary aapko in terms ke full forms aur unka real-world meaning simple words mein batati hai.*

## 2. Alphabetical IT Glossary Reference

### A - D
- **AD (Active Directory):** Microsoft's directory service database for managing user, computer, and network resources.
- **APIPA (Automatic Private IP Addressing):** A self-assigned local IP address (`169.254.x.x`) used when a device cannot reach a DHCP server.
- **ARP (Address Resolution Protocol):** Maps Layer 3 IP addresses to Layer 2 physical MAC addresses on a local subnet.
- **BIOS (Basic Input/Output System):** Motherboard firmware used to initialize hardware and run POST checks during boot.
- **BSOD (Blue Screen of Death):** A system crash screen displayed when the Windows kernel encounters a critical halt condition.
- **CAB (Change Advisory Board):** A committee that reviews and approves modifications to IT infrastructure.
- **CIDR (Classless Inter-Domain Routing):** A standard method for allocating IP addresses and indicating network masks.
- **CMDB (Configuration Management Database):** A database that tracks IT asset items (CIs) and their relational mappings.
- **DHCP (Dynamic Host Configuration Protocol):** Automatically leases IP address configurations to client network adapters.
- **DISM (Deployment Image Servicing and Management):** A tool used to check and repair local Windows system image files.
- **DNS (Domain Name System):** Resolves human-readable domain hostnames (e.g., `google.com`) to IP addresses.
- **DORA:** The four-step DHCP address leasing sequence: Discover, Offer, Request, Acknowledge.

### E - L
- **EDR (Endpoint Detection and Response):** Real-time endpoint monitoring software designed to detect and isolate cyber threats.
- **ESD (Electrostatic Discharge):** The sudden transfer of static electricity, which can damage hardware.
- **FQDN (Fully Qualified Domain Name):** The absolute domain address of a host system (e.g., `workstation01.corp.local`).
- **GPO (Group Policy Object):** Administrative settings linked to AD containers to configure client settings.
- **HAL (Hardware Abstraction Layer):** An OS layer (`hal.dll`) that hides physical hardware differences from the kernel.
- **HTTP/HTTPS:** Protocols used to transmit web data. HTTPS encrypts traffic using SSL/TLS protocols.
- **ICMP (Internet Control Message Protocol):** Protocol used by utilities like `ping` to send error and status packets.
- **Intune:** Microsoft's cloud-based Mobile Device Management (MDM) endpoint administration platform.
- **ITIL (Information Technology Infrastructure Library):** A framework of best practices for delivering IT services.
- **KMS (Key Management Service):** Local client activation server used to activate Windows volume keys over LAN.
- **LDAP/LDAPS:** Protocol used to query directory service objects. LDAPS encrypts queries using SSL/TLS.
- **LSDOU:** The processing order of Group Policies: Local, Site, Domain, Organizational Unit (OU).

### M - Z
- **MAC (Media Access Control) Address:** A unique 48-bit hardware address burned into a network card (NIC) by the manufacturer.
- **MFA (Multi-Factor Authentication):** A security verification method requiring multiple credentials to access accounts.
- **MSI (Microsoft Installer):** Standard database file format for installing software packages in Windows.
- **M365 (Microsoft 365):** Microsoft's cloud subscription suite managing user mailboxes, Office apps, and Entra identities.
- **NTP (Network Time Protocol):** Synchronizes local computer system clocks with network time servers.
- **OST (Offline Storage Table):** A local cache copy of a user's exchange mailbox data. Safe to delete.
- **OU (Organizational Unit):** A logical container folder in Active Directory used to organize resources and link GPOs.
- **POST (Power-On Self-Test):** Hardware checks performed by the BIOS/UEFI on startup.
- **PST (Personal Storage Table):** A local personal archive file used to store email folders. Data is lost if deleted.
- **RDP (Remote Desktop Protocol):** Protocol used to access GUI screens of remote workstations.
- **SFC (System File Checker):** Scans and repairs corrupted operating system files using local cache files.
- **SLA (Service Level Agreement):** Agreements defining committed response and resolution time targets for support tickets.
- **SMB (Server Message Block):** Protocol used to share folders and files in Windows local networks.
- **SOP (Standard Operating Procedure):** Detailed step-by-step documentation for completing routine tasks.
- **TPM (Trusted Platform Module):** A hardware security chip on the motherboard that secures cryptographic keys.
- **UPN (User Principal Name):** The internet-style login username for AD accounts (e.g., `user@company.com`).
- **VPN (Virtual Private Network):** Creates an encrypted tunnel to extend secure connections over public networks.
- **WinRE (Windows Recovery Environment):** A recovery platform used to run startup diagnostic tools.
- **WSUS (Windows Server Update Services):** Local update server distributing approved patches to corporate clients.
